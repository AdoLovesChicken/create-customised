# Changelog: 0.8.0b Update

## Feature Changes
- Removed all features from the following mods:
  - Artifacts
  - Accessories
  - Accessories Compatibility Layer
  - Vanilla Zoom
  - Create & FD Dough
  - Toni's Immersive Lanterns
  - Create: Fishery Industry
  - Create: Hypertubes
  - Create: Easy Stone Generators
  - Create: Integrated Farming
  - Create: Diamond & Factory
- Jetpacks now work in their respective Curios slots
- Pressing Left Alt no longer opens a fake spyglass zoom
- Spyglasses can now be bound to the belt slot in Curios, can be hotkeyed (default C), and can be zoomed using the scroll wheel
- Crushing recipes for Create Easy Structures tracks are now even
- Added numerous functional decorative blocks related to trains
- When not in full screen, the window bar is dark
- Unified FD's and Create's doughs into one
